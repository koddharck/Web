## 🎬 Animation Examples

### Fade In
```typescript
initial={{ opacity: 0 }}
animate={{ opacity: 1 }}
transition={{ duration: 0.6 }}
```

### Slide Up
```typescript
initial={{ opacity: 0, y: 30 }}
animate={{ opacity: 1, y: 0 }}
```

### Scale on Hover
```typescript
whileHover={{ scale: 1.05 }}
```

### Staggered List
```typescript
variants={containerVariants}
initial="hidden"
whileInView="visible"
```

See `ANIMATIONS_GUIDE.md` for more examples and customization options.

## 🔧 Troubleshooting

### Port 3000 in use?
```bash
npm run dev -- --port 3001
```

### Styles not showing?
- Restart dev server
- Clear browser cache
- Check `tailwind.config.js`

### Build errors?
```bash
npm cache clean --force
rm -r node_modules
npm install
npm run build
```

### Dark mode not working?
- Check localStorage settings
- Verify `ThemeContext.tsx`
- Check `tailwind.config.js` darkMode

See `SETUP_GUIDE.md` for more troubleshooting.

## 🎨 Customization Examples

### Change Primary Color
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: {
    500: '#YOUR_COLOR_HEX',
    // ... other shades
  }
}
```

### Change Font
Edit `src/styles/globals.css` and `tailwind.config.js`:
```css
@import url('https://fonts.googleapis.com/css2?family=YOUR_FONT&display=swap');
```

### Add Animation Speed
Edit any motion component:
```typescript
transition={{ duration: 1.0 }} // was 0.3
```

### Add New Page
1. Create `src/pages/NewPage.tsx`
2. Add route in `src/App.tsx`
3. Add nav link in `src/components/Navbar.tsx`
